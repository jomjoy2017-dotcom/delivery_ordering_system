# Delivery Ordering System

ระบบแอปพลิเคชันสั่งสินค้าเดลิเวอร์รี่ รองรับลูกค้าและผู้ขาย มีระบบสมัครสมาชิก สะสมแต้ม จัดการตะกร้าสินค้า และระบบหลังบ้านสำหรับผู้ขายที่สามารถรันรหัสสินค้าอัตโนมัติ ออกรายงานยอดขาย กำไร-ขาดทุน และปริ้นใบสั่งซื้อรูปแบบ PDF ได้

## Features
- ระบบสมาชิก: ลูกค้าสามารถสมัครสมาชิก (ชื่อ นามสกุล เบอร์โทร ที่อยู่) เข้าสู่ระบบและสะสมแต้ม (10 แต้มต่อการสั่งซื้อ)
- ระบบสั่งซื้อสินค้า (ลูกค้า): แสดงเมนูสินค้า เลือกจำนวนได้หลายรายการ ยืนยันการสั่งซื้อพร้อมแสดงยอดรวม และสถานะรอจัดส่ง
- ระบบจัดการสินค้า (ผู้ขาย): เพิ่ม/ลบ/แก้ไขข้อมูลสินค้า (รองรับคำอธิบาย 50 คำ) กำหนดราคาต้นทุน ราคาขาย และอัปโหลดภาพลงโฟลเดอร์ images
- รหัสสินค้าอัตโนมัติ: ระบบหลังบ้านสร้างรหัสสินค้าเรียงตามลำดับ (เช่น sl0001, sl0002) อัตโนมัติ
- ระบบแสดงผลสินค้า: ค้นหาด้วยรหัสสินค้า แสดงหน้าละ 20 รายการ พร้อมระบบแบ่งหน้า (Pagination)
- ระบบออเดอร์ (ผู้ขาย): ตรวจสอบรายการสั่งซื้อที่เข้ามา (แสดงรายการกาแฟ/สินค้า ชื่อลูกค้า เบอร์โทร ที่อยู่) และกดยืนยันออเดอร์
- ระบบรายงานและปริ้น PDF: แสดงรายงานประจำวัน/เดือน/ปียอดขายและกำไรขาดทุน และมีหน้าปริ้น PDF ภาษาไทย (ใช้ CSS Print) สำหรับใบสั่งซื้อ
